Projects

Introduction

Welcome! In this blog post, I’m going to show you all of the projects I’ve been working on and the progress I’ve made. Stay tuned to find out more!

Zinix OS

A Unix-like operating system built from scratch was implemented. Beside some basic OS functionalities, such as basic device support, interrupts, memory segmentation+paging, filesystem, multiple terminals, systemcalls and Round-Robin scheduler, advanced functionalities were also implemented, such as memory allocation, signals, ATK disk driver, and speaker driver.

Crime Map UIUC

We implemented a database containing past crime logs around UIUC and allowed users to do basic CRUD commands on the database through our web page. We also integrated Google Map API to show different crimes on a map to visualize them for users. Web page is built with ReactJS and NodeJS, database is implemented on GoogleCloud MySQL server.

Database system for medical appointments

We used C++ to build a B+ Tree as the base for our database system. To make it more user-friendly, we implemented a UI instead of relying on terminal commands. Additionally, we imitated a SQL style query language to make our system easier to learn for those familiar with SQL.

Conclusion
